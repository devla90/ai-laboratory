# Moderator Guide Format

## Cuando se Activa
- Generar guía del moderador para un lab
- Revisar o actualizar una guía existente

## Estructura de la Guía

### Encabezado
- Título del lab, duración, audiencia, objetivo principal
- Checklist de preparación previa

### Por Cada Sección
1. **Título y timing** (ej: "Sección 3: Cómo funciona la IA - 7 min")
2. **Puntos clave**: explicación detallada de cada concepto
3. **Guion sugerido**: frases textuales entre comillas que el moderador puede usar
4. **Analogías**: detalladas con contexto y desarrollo
5. **Preguntas del público**: P/R anticipadas (3-5 por sección)
6. **Transición**: frase para conectar con la siguiente sección

### Tono
- Calmado, sin prisa
- Explicaciones claras y progresivas
- Lenguaje accesible (evitar jerga técnica sin explicar)
- Las analogías deben conectar con experiencias cotidianas del equipo

### Secciones Interactivas
- Indicar cuánto tiempo dar para que respondan
- Sugerir cómo redirigir si nadie responde
- Sugerir cómo manejar respuestas incorrectas sin invalidar
